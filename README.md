# android-repository

This is a set of scripts for creating a mirror of Android Repository, http://dl-ssl.google.com/android/repository/*

Works on Linux Mint Maya/13 (Ubuntu 10.04.x)

## Prerequisites

```
sudo aptitude install libsaxon-java default-jre
```
